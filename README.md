Descripcion del problema : La administración de gastos comunes en edificios residenciales se realiza frecuentemente de forma manual o mediante herramientas desarticuladas, lo que genera errores en el prorrateo, falta de transparencia en la información financiera para los residentes y dificultades para el seguimiento oportuno de la morosidad.

Solución : crear un sistema que automatiza el cálculo de gastos comunes (prorrateo), gestiona el control de morosidad con alertas automáticas y proporciona una plataforma de consulta en tiempo real para residentes y administradores, asegurando la integridad de los datos
